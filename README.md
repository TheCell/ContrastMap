# contrastmap

> studio web mobile project of emmen

It was created over 3 Weeks by @lauragyr and myself

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
